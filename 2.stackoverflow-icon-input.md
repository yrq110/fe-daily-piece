## stackoverflow-icon-input

* date: `2017/04/20`
* source: [Stack Overflow](http://stackoverflow.com/)
* demo: [codepen demo](https://codepen.io/yrq110/pen/dWGrNe?editors=1100)

### HTML
```html
<div id="search">
  <button id="btn" class="in"></button>
  <input name="s" type="text" placeholder="search"/>
</div>
```

### CSS
```css
#search {
    width: 500px;
}

#btn {
    display: inline-block;
    cursor: pointer;
    position: absolute;
    left: 465px; top: 10px;

    background-color: rgb(0,149,255);
    border: none;

    visibility: visible;

.in {
    visibility: hidden;

```

### JS
```js
var btn = document.getElementById("btn");
var input = document.getElementsByName("s")[0];

var a = false;

input.onfocus = function () {
  btn.className = "out";
};

input.onblur = function(){
  if(a == false){
    btn.className = "in";
  }
};

btn.onmouseenter = function () {
  a = true;
};

btn.onfocus = function () {
  a = false;
  input.focus();
};

```
